# Property Lookup Agent

A lightweight, API‑driven property lookup tool for Wake County, NC. Users enter an address, select from an autocomplete dropdown, and receive a structured summary of parcel characteristics, ownership, valuation, and permit history.

## Features

- **Address‑based search with autocomplete**
- **PIN‑based lookup for accuracy**
- **Clean, sectioned HTML output**
- **Normalized datasets for fast filtering**
- **n8n workflow powering the backend**
- **GitHub Pages front‑end (in progress)**

## Repository Structure

- **/data:** cleaned datasets  
- **/raw:** original source files  
- **/workflows:** n8n workflow JSON  
- **/docs:** architecture and design notes
